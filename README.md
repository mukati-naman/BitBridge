# 🚀 Online Coding Platform

A full-stack **online coding platform** that allows users to solve coding problems, run code using Judge0 API, and track their submissions — similar to platforms like LeetCode and CodeChef.

---

## 🌟 Features

### 👨‍💻 User Features
- 🔐 Secure Authentication (Signup/Login)
- 🧠 Solve coding problems with multiple difficulty levels
- ⚡ Run code in multiple languages using Judge0
- 📊 View submission history
- ⏱️ Real-time code execution results

### 🛠️ Admin Features
- ➕ Create new coding problems
- ✏️ Update existing problems
- ❌ Delete problems
- 🔒 Protected admin routes

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```
git clone https://github.com/mukati-naman/BitBridge.git
cd BitBridge
```

### 2️⃣ Setup Backend
```
cd backend
npm install
npm start
```

### 3️⃣ Setup Frontend
```
cd frontend
npm install
npm run dev
```

## 🔄 How It Works

1. User selects a coding problem
2. Writes code in the editor
3. Code is sent to backend
4. Backend sends it to Judge0 API
5. Execution result is returned and displayed
6. Submission is stored in database
